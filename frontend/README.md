# Forever Frontend

A modern and responsive e-commerce frontend built with React and Vite. The application provides a smooth shopping experience with product browsing, filtering, searching, cart management, and responsive design.

    Note: This is a dummy project created for learning, practice, and portfolio purposes. Products, prices, images, and order functionality are for demonstration only and do not represent a real online store.

## Features

- Product listing and collections
- Search functionality
- Product filtering and sorting
- Product details page
- Shopping cart management
- Responsive design for mobile and desktop
- Related products section
- Newsletter subscription section

## Tech Stack

- React.js
- Vite
- React Router DOM
- Context API
- Tailwind CSS

## Build for Production

```bash
npm run build
```
